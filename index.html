HTML

TAGs Resumo
<!DOCTYPE> define quais especificações que o documento segue
<b> Texto em negrito
<div> Divisão em bloco
<em> Texto enfatizado
<h1> à <h6> Títulos onde cada valor numérico define tamanho e importância
<head> Cabeçalho
<body> Corpo da Página
<i> Texto em itálico
<img> Tag para adicionar imagem
<p> Parágrafo
<strong> Texto Grande
<style> Define estilo
<table> Tag de tabela
<tr> Linha da tabela
<th> Coluna da Tabela
<title> Título da Página
<meta> Define um meta-informação
<link> Link de recursos
<br> Quebra a linha

CSS

Propriedades Resumo
color: muda a cor do texto
font-family: define uma lista de fontes para ser utilizada no elemento
font-size: muda o tamanho do texto
background-color: muda a cor de fundo
width: define largura
height: define altura
margin: define o tamanho da margem
padding: define a "margem" interna
rgb() maneira de definir cor
rgba() maneira de definir cor e transparência
text-align: alinha o texto na posição definida
